TASK6 | ML选股模型 · Top30季度策略 · 回测对比 | 80只股票 · 10因子 · 5年数据

基于机器学习的交易策略

核心理念：利用机器学习算法从历史数据中挖掘因子与未来收益之间的非线性关系，构建预测模型对股票未来收益率进行排序，然后基于排序结果构建选股策略——例如每个季度选择模型预测收益最高的30只股票进行等权投资，定期调仓，并对比策略收益与市场平均收益。

✓ 策略优点

  • 优非线性建模能力：机器学习模型（特别是树模型）能捕捉因子与收益间的复杂非线性关系，突破传统线性回归的局限
  • 优多因子融合：可同时纳入技术面、基本面、情绪面等数十个因子，自动学习因子权重
  • 优自适应更新：模型可定期重新训练，适应市场环境变化
  • 优特征重要性：随机森林等模型可输出因子重要性排序，辅助因子筛选
  • 优规模可扩展：策略可覆盖全市场数千只股票，人工难以处理的维度机器轻松应对
  • 优客观纪律性：模型决策不受情绪影响，严格执行策略规则

✗ 策略缺点

  • 缺过拟合风险：模型可能在训练集表现优异但在测试集失效，需要严格的样本外验证
  • 缺数据依赖：策略效果高度依赖数据质量，噪声数据或数据缺失会严重影响预测
  • 缺市场环境敏感：在结构性变化（如金融危机、政策转向）时，历史规律可能失效
  • 缺可解释性弱：复杂模型（如深度学习）的决策逻辑不透明，难以向投资者解释
  • 缺交易成本：定期调仓产生较高的交易成本和冲击成本，可能侵蚀超额收益
  • 缺因子衰减：有效因子可能随时间衰减，需要持续挖掘新因子

⚙ 策略构建流程

1. 数据准备 → 多股票多季度面板数据
2. 因子构建 → 技术因子 + 财务因子
3. 应变量定义 → 下季度收益率
4. 训练测试划分 → 按时间前向分割
5. 模型训练 → LR / DT / RF 对比
6. 选股策略 → 预测排序 → Top30
7. 回测评估 → 累积收益/夏普/回撤
8. 策略对比 → 各模型 vs 市场基准

关键设计：训练集和测试集按时间划分（前14个季度训练，后5个季度测试），模拟真实的"用历史预测未来"场景，避免数据泄露。

自变量因子与应变量定义

在量化交易的机器学习模型中，自变量（因子/特征）是用于预测的输入变量，应变量（目标变量）是模型需要预测的输出。因子设计是ML量化策略的核心环节，好的因子是模型成功的基础。

自变量因子定义（10个因子）

因子名称类型计算方式经济含义
波动率(volatility)技术日收益率标准差×√252衡量股票风险水平，高波动=高风险
动量(momentum)技术过去N期收益率趋势延续效应，强者恒强
成交量变化(volume_change)技术成交量同比增长率市场关注度变化，放量往往伴随趋势
均线趋势(ma_trend)技术价格/MA20 - 1判断多空趋势，价格在均线上方=多头
换手率(turnover)技术成交量/流通股本流动性指标，高换手=交易活跃
市盈率(pe_ratio)财务股价/每股收益估值水平，低PE可能被低估
市净率(pb_ratio)财务股价/每股净资产估值水平，低PB=安全边际高
ROE财务净利润/净资产盈利能力核心指标，高ROE=优质公司
营收增长率(revenue_growth)财务(本期营收-上期)/上期成长性指标，高增长=发展潜力大
净利率(net_profit_margin)财务净利润/营收盈利质量，高净利率=定价能力强

应变量定义

应变量: next_return = 下一季度收益率 (%)
计算方式: next_return = (P(t+1) - P(t)) / P(t) × 100
任务类型: 回归 (Regression)
预测目标: 对所有股票的下季度收益率进行排序

为什么用下季度收益率作为应变量？

  • 直接对应投资收益，预测值有明确的经济含义
  • 回归任务可以保留收益率大小的排序信息（比分类更精细）
  • 基于预测值排序选股，天然适合"选Top N"的策略框架
  • 可灵活调整持仓数量（Top 10/20/30/50）进行敏感度分析

为什么按时间划分训练/测试集？

  • 模拟真实投资场景：用过去数据训练模型，预测未来收益
  • 避免数据泄露：测试集在训练集之后，不存在未来信息泄露
  • 前向验证(walk-forward)：更接近实盘交易的决策过程

结论与分析

  • 随机森林策略最优：总收益8.63%，超额收益6.20%，夏普比率2.24，最大回撤仅0.47%，季度胜率80%。集成模型通过Bagging+随机特征选择有效降低了方差。
  • 线性回归次之：总收益5.92%，超额3.50%，夏普1.05。虽然R²接近0（因子与收益线性关系弱），但排序选股仍能创造超额收益。
  • 决策树表现最差：总收益-3.64%，跑输市场。单棵决策树容易过拟合，在测试集上泛化能力差。
  • 市场基准：总收益2.42%，年化1.93%，作为被动投资的对比基准。
  • 最重要因子：营收增长率、ROE、波动率排名前三，说明成长性、盈利能力和风险水平是预测未来收益的关键因子。
  • 策略启示：即使模型R²较低，只要排序能力足够强，选股策略仍能产生显著超额收益。ML选股的核心不在于预测收益的绝对值，而在于相对排序的准确性。

Python 实现代码

以下代码实现完整的ML选股策略流程：数据生成→因子构建→模型训练→选股策略→回测评估。

import numpy as np
import pandas as pd
from sklearn.tree import DecisionTreeRegressor
from sklearn.ensemble import RandomForestRegressor
from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import StandardScaler

# 1. 加载数据 (80只股票×20季度, 10个因子)
df = pd.read_csv('ml_stock_data.csv')

# 2. 定义自变量(因子)和应变量
feature_cols = ['volatility', 'momentum', 'volume_change',
                'ma_trend', 'turnover',           # 技术因子
                'pe_ratio', 'pb_ratio', 'roe',
                'revenue_growth', 'net_profit_margin']  # 财务因子
target_col = 'next_return'  # 应变量: 下季度收益率

# 3. 按时间划分训练集/测试集
train_quarters = list(range(0, 14))  # Q1-Q14训练
test_quarters = list(range(14, 19))  # Q15-Q19测试

train_df = df[df['quarter'].isin(train_quarters)]
test_df = df[df['quarter'].isin(test_quarters)]

X_train = train_df[feature_cols].values
y_train = train_df[target_col].values
X_test = test_df[feature_cols].values
y_test = test_df[target_col].values

# 标准化 (线性回归需要)
scaler = StandardScaler()
X_train_s = scaler.fit_transform(X_train)
X_test_s = scaler.transform(X_test)

# 4. 构建并训练模型
models = {
    '线性回归': LinearRegression(),
    '决策树': DecisionTreeRegressor(max_depth=6, random_state=42),
    '随机森林': RandomForestRegressor(
        n_estimators=100, max_depth=8, random_state=42),
}

for name, model in models.items():
    if name == '线性回归':
        model.fit(X_train_s, y_train)
        test_df[f'pred_{name}'] = model.predict(X_test_s)
    else:
        model.fit(X_train, y_train)
        test_df[f'pred_{name}'] = model.predict(X_test)

# 5. 构建选股策略: 每季度选Top30
TOP_N = 30

for name in models:
    quarterly = []
    for q in test_quarters:
        q_data = test_df[test_df['quarter'] == q]
        # 按预测收益率排序,选Top30
        top = q_data.nsmallest(TOP_N, f'pred_{name}',
                              keep='all').head(TOP_N)
        top = q_data.sort_values(f'pred_{name}',
                                 ascending=False).head(TOP_N)
        strat_ret = top['next_return'].mean()
        market_ret = q_data['next_return'].mean()
        quarterly.append({'strategy': strat_ret,
                          'market': market_ret})

    # 6. 计算累积收益和核心指标
    strat = pd.DataFrame(quarterly)
    strat['cum_strategy'] = (1+strat['strategy']/100).cumprod()-1
    strat['cum_market'] = (1+strat['market']/100).cumprod()-1

    total = strat['cum_strategy'].iloc[-1] * 100
    market = strat['cum_market'].iloc[-1] * 100
    sharpe = (strat['strategy'].mean() /
              strat['strategy'].std()) * np.sqrt(4)
    win_rate = (strat['strategy'] > strat['market']).mean() * 100

    print(f"{name}: 总收益={total:.2f}%, "
          f"超额={total-market:.2f}%, 夏普={sharpe:.2f}")
